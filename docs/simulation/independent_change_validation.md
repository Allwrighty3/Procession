# Independent-change contingency validation

This experiment compares reactive, outcome-only adaptive, and locally contingent adaptive behavior under reliable and noisy movement, with and without environmental intake changes that occur independently of the entity's actions.

The world stores current state only. The entity retains event-specific, decaying traces for emitted action, sensed displacement, and immediate local intake change.

This branch exists only to trigger and capture the validation run for PR #9.
